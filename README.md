# Grokログ抽出ツール

Grokのダウンロードデータから会話部分だけを抽出し、HTML形式で見やすく整理するPythonスクリプトです。

HTMLは各チャットごとに `チャットタイトル名.html` として出力されます。

txt / json は補助的な出力です。

## 事前準備

### Grokログのダウンロード

- https://accounts.x.ai/data からログをダウンロードして解凍

### Pythonのインストール

- https://www.python.org/ からインストール

### スクリプトの配置

- `prod-grok-backend.json` があるフォルダに `grok_log_export.py` を保存
- 保存時、テキストエディタで「ファイルの種類」を「すべてのファイル」に設定
- 拡張子 `.py` を必ず指定

## 実行手順

### 保存先パスの確認

例：

```
C:\Users\ユーザー名\Downloads\解凍したログのフォルダ\ttl\30d\export_data\英数字の羅列\
```

### コマンドプロンプトを開く

- Windows検索で `cmd` と入力 → Enter

### コマンドを実行

```
cd "C:\Users\ユーザー名\Downloads\解凍したログのフォルダ\ttl\30d\export_data\英数字の羅列\"
python grok_log_export.py
```

## 出力内容

- 各チャットごとに `チャットタイトル名.html` を生成
- Grok履歴画面と同じ順序（新しい→古い）で保存
- ピン留めは考慮されません

## 注意事項

- タイムスタンプは日本時間（JST）として表示されます
- Grokの回答ではJST(+9時間)ではないとされますが、確認上はJSTで出力されている可能性が高いです
- 処理結果に不完全な部分がある場合があります
- 必要に応じて各自で調整してください
